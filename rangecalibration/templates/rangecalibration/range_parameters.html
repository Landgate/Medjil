Calibration Range Parameters

The range calibration parameters consist of height differences between adjacent pins (e.g., 1-2, 2-3, etc.). Each height difference is an averaged value from several independent surveys conducted during a specific month (e.g., Jan, Feb, ...).

Laboratory calibrated state-of-the-art invar stave together with high precision digital level are used to measure the range height differences. The measurements are regularly carried out in order to monitor the stability of the Range calibration site and are used to update the range parameters.
